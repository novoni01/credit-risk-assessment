TO USE KAGGLE API
In order to use Kaggle api you need to 
    1. Create a free kaggle account
    2. Click on your profile picture
    3. Download "Legacy API Key" that should produce a 'kaggle.json' file in your download folder
    4. Go to your OS:Users:'YourProfile'
    5. Create a folder named '.kaggle'
    6. Drag and drop the 'kaggle.json' into the '.kaggle' folder
